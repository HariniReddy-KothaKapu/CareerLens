# CareerLens 🔍  
### ML-Based Career Skill Gap Analyzer

CareerLens is an **ML-powered web application** that helps users identify **skill gaps across 100+ IT and software engineering roles**. It analyzes a user’s existing skills and compares them with industry-required skills to recommend **missing skills, learning paths, and career improvements**.

The system leverages **Machine Learning and Natural Language Processing (NLP)** to intelligently match user skills with job role requirements.

---

## 📌 Problem Statement

Students and job seekers often struggle to understand:
- What skills are required for a specific role  
- What skills they currently lack  

CareerLens solves this by:
- Automatically analyzing user skills  
- Identifying **missing skills for target roles**  
- Providing **structured and level-based recommendations**

---

## ⚙️ Working of the System

1. User registers/logs into the platform  
2. User uploads their resume  
3. User selects a desired career role  
4. NLP model extracts and processes skills  
5. System compares user skills with dataset  
6. Missing skills are identified  
7. Recommendations are generated (Beginner → Advanced)  
8. Personalized course suggestions are displayed
